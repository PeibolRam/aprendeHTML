Titulos

Otros elementos HTML muy utilizados son para indicar los títulos, contamos con los elementos:

<h1>

<h2>

<h3>

<h4>

<h5>

<h6>

El título de mayor nivel es <h1>, es decir el que tiene normalmente una fuente de mayor tamaño.

Mira el siguiente ejemplo: 

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Titulos</title>
</head>
<body>
    <h1>h1</h1>
    <h2>h2</h2>
    <h3>h3</h3>
    <h4>h4</h4>
    <h5>h5</h5>
    <h6>h6</h6>
</body>
</html>
