Imagenes

Para insertar una imagen dentro de una página debemos utilizar el elemento HTML <img>, la misma no tiene una marca de finalización

Generalmente, la imagen se encuentra en el mismo servidor donde se almacenan nuestras páginas HTML. Los formatos clásicos son los archivos con extensiones gif, jpg y png.

La sintaxis de esta etiqueta es:

<img src="images.jpg" alt="Codeando html"> 

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Hypervinculos</title>
</head>
<body>
<img src="images.jpg" alt="Codeando html"> 
</body>
</html>
